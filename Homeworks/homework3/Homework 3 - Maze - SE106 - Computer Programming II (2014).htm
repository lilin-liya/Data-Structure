Homework 3: Maze

due on 2014-10-8 23:59 UTC+8 Download Starter Files

Introduction

In homework 3, you need use recursive find a path to get out a maze.

Format

As you can see from download files, Maze.cpp is empty except comments, you need write all code by yourself. However, you must follow our I/O format.
In file testmap_1.txt:
4 6
+--+
|S |
+- |
|- |
E  |
+--+
 The first line is two number, tell you the length and width of the maze, this maze is a 4*6 size maze.
And 'S' is your start point, 'E' is the end, '|' '-' '+' are walls, the work you need to do is to find a path to the end, and output the path to terminal. For this maze, you should output:
testmap_1: (2,1) (2,2) (2,3) (2,4) (1,4) (0,4) testmap_1 is maze name, (2,1) means move to x=2, y=1 in this maze.
There are three test maze, and we won't test other maze.
You can enter make to generate a execute file, and when you run it, it can output to terminal: testmap_1: ...
testmap_2: ...
testmap_3: ... If you want to add other .cpp or .h file, don't forget to modify Makefile

Socre

You will get full credit if you output correct path and use recursive

Late committing will be punished
